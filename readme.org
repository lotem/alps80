** 鍵盤佈局

75% 類 JIS 佈局，共 80 鍵，含 CAN、GRAPH、LF 等特殊按鍵。
左上角 2 顆、左下角 1 顆及 7U 空格鍵帽缺失。

** 矩陣
兩組排線 CN1（10 針）、CN2（11 針），編號 P1-P10，P11-P21。

P1-P2 僅連接右上角三角符號鍵 SW15，併入矩陣，令 P1 = C10，P2 = R7。

左右 Shift（SW58、SW71）共用矩陣座標 (R3, C0)。
擬將 SW71 飛線改到 (R3, C10) 位置，因不願破壞原裝電路而作罷。
